Home page

Four entries:
Study materials, Campus Encyclopedias, Utilities and About

  • “凌波微光” is a campus guide program which is implemented in WeChat program.

  • It provides encyclopedia, news, personal class schedule, learning materials etc. for all staff and students.

  • Implemented API interface with Thinkphp framework and use Python to collect courses information

Background

When I was in college, I found that:

  • Information in college is scattered

  • Classes vary a lot weekly

  • Some of the electives are not helpful
